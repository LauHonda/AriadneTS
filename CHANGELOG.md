# Changelog

## 0.2.0 - Unreleased

- Added the synchronous engine-independent JSON host bridge.
- Added Unity host-handler registration and script invocation APIs.
- Added Unity script diagnostic modes and optional script log files.
- Increased the native runtime ABI version to 4.

## 0.1.0 - Unreleased

- Added the cross-engine QuickJS runtime C ABI.
- Added the Unity UPM host and native plugins.
- Added signed single-file script packages for Addressables.
- Added lifecycle dispatch, Promise jobs, stateful runtime switching, and
  runtime resource limits.
