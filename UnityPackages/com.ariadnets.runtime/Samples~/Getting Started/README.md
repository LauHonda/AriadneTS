# AriadneTS Getting Started Sample

This sample contains a small C# host bridge for the default TypeScript
template generated by **Tools > AriadneTS > Script Tools**.

## Use It

1. Import this sample from Unity Package Manager.
2. Open **Tools > AriadneTS > Script Tools**.
3. Initialize the TypeScript project.
4. Select a private key and build `typescript-package.bytes`.
5. Click **Create Or Update Runtime Host In Scene**.
6. Keep the generated `AriadneTsDemoHostBridge`, or replace it with
   `AriadneTsSampleHostBridge` from this sample.
7. Enter Play Mode.

The default TypeScript template calls `demo.getPlayer` during `start()`.
`AriadneTsSampleHostBridge` handles that call and returns JSON to TypeScript.

For production, keep private keys outside source control and ship only the
public key with the Unity application.
