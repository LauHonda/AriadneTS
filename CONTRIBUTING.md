# Contributing to AriadneTS

Thanks for helping improve AriadneTS.

## Development

```sh
./Tools/test_all.sh
./Tools/build_unity_plugins.sh
```

Keep public QuickJS implementation types behind the C ABI. Unity-specific
download, cache, and version-selection logic belongs outside the runtime.

## Pull Requests

- Keep changes focused.
- Add or update tests for behavioral changes.
- Run the complete test suite before opening a pull request.
- Document public API and package-format changes.

By contributing, you agree that your contributions are licensed under MIT.
